**Button** — the brand's call-to-action; use `primary` for the main action on any section, `secondary` on dark grounds, `navy` on light grounds.

```jsx
<Button variant="primary" as="a" href="/enrol" icon={<span>›</span>}>Enrol Now</Button>
<Button variant="secondary">Learn More</Button>
```

Variants: `primary` (crimson fill + offset gold outline), `secondary` (ghost, gold border — dark grounds only), `navy` (solid navy — light grounds), `ghost` (text + gold underline). Sizes: `sm` `md` `lg`. Labels are always Cinzel, uppercase, letter-spaced — keep them short (1–3 words).
