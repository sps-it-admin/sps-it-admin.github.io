**StatBlock** — a single figure with a gold left-rule and Cinzel label. Use in rows of 2–4 for key facts (founded, year levels, location).

```jsx
<StatBlock value="1999" label="Founded" />
<StatBlock value="7 & 8" label="Year Levels" tone="dark" />
```

Set `tone="dark"` on navy grounds. Group in a flex/grid row with `gap`.
