**PillarCard** — white bordered feature card (crimson icon, Playfair title, Lora body) for value/feature grids. Lifts with a soft shadow on hover.

```jsx
<PillarCard icon={<Cross size={22} />} title="Catholic Faith">
  Every subject is taught within the framework of the Catholic Faith.
</PillarCard>
```

Square corners, no rounding. Use in a 2- or 4-column grid with a `gap`.
